# hxtpm Test Library

**Package:** `com.hyperxtalk.library.test`
**Type:** Script Library
**Author:** HyperXTalk
**License:** gpl-3.0
**Repository:** https://gist.github.com/emily-elizabeth/e0f8c61b7ed48b6b422f5cd113b20cbb

Minimal test library for verifying the hxtpm package manager install pipeline. Call hxtpmTestPing() after install to confirm the library is live.

## Installation

```
hxtpmInstall "com.hyperxtalk.library.test"
```
